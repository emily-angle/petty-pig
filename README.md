
# petty-pig - a website admin manager system

petty-pig 是一个基于 Create React App的脚手架 建立起来的网站后台管理程序

## Getting Started


```bash

git clone  https://github.com/emily-angle/petty-pig.git
cd petty-pig
npm install # or yarn
```
### Preview

```bash
npm start
```
